Berdasarkan deskripsi item koleksi museum berikut yang berjudul "Ani-ani", mohon buatkan sejumlah 2 pertanyaan pilihan ganda.

Format jawaban Anda sebagai file JSON berisi daftar soal pilihan ganda dengan atribut "question" (pertanyaan), "choices" (pilihan jawaban), dan "answer" (indeks jawaban benar). Jangan sertakan teks pengantar, penjelasan, atau format markdown apa pun di luar struktur JSON.

Contoh format:
[
  {
    "question": "Pertanyaan 1?",
    "choices": [
        "A. Pilihan 0",
        "B. Pilihan 1",
        "C. Pilihan 2",
        "D. Pilihan 3"
    ],
    "answer": 0
  },
  {
    "question": "Pertanyaan 2?",
    "choices": [
        "A. Pilihan 0",
        "B. Pilihan 1",
        "C. Pilihan 2",
        "D. Pilihan 3"
    ],
    "answer": 3
  }
]
